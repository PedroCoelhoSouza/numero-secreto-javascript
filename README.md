# Jogo do Número Secreto

Este projeto é um jogo simples desenvolvido para praticar lógica de programação com **JavaScript**, **HTML** e **CSS**.

O objetivo do jogo é adivinhar um número secreto gerado aleatoriamente pelo sistema.
O jogador recebe dicas informando se o número digitado é maior ou menor que o número secreto até acertar.

## Acesse o projeto

Você pode testar o jogo online:

🔗 https://numerosecreto-two-dun.vercel.app

## Tecnologias utilizadas

* HTML
* CSS
* JavaScript

## Como jogar

1. O sistema gera um número secreto aleatório.
2. O jogador digita um número no campo de entrada.
3. O jogo informa se o número secreto é **maior** ou **menor** que o valor digitado.
4. Continue tentando até acertar o número secreto.

## Conceitos praticados

Neste projeto foram utilizados conceitos importantes de programação:
